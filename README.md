### 프로젝트 설명

본 프로젝트를 통해 아래와 같은 경험과 지식을 쌓고 있습니다.

- SpringBoot 기반의 애플리케이션 기획 배포 경험
- AWS 클라우드 환경 기반 CICD 무중단 배포 인프라 구축 경험
- JPA, Hibernate를 사용한 도메인 설계 경험
- 성능 향상을 위한 고민과 도전 경험
- Test 코드 작성을 통한 안정성 확보

## 기술 스택

- Java 17
- Gradle
- Spring Boot 3.2.5
- Spring Data JPA
- Spring Security
- QueryDSL
- PostgreSQL
- Junit5, Mockito
- AWS(EC2, RDS, S3)
- GitHub Actions
